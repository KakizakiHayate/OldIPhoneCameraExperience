# CameraScreen レイアウト仕様

## 写真モード

```
┌──────────────────────┐
│                      │
│ [4:3]          [⚡] │ ← ボタンが黒帯の上に乗っている
│──────────────────────│ ← 黒帯とプレビューの境界
│                      │
│                      │
│   カメラプレビュー   │ ← アスペクト比に応じて変化（3:4 / 1:1 等）
│                      │
│                      │
│                      │
│──────────────────────│
│     [0.5] [1x]       │ ← ズームプリセット
│ [□] [●シャッター] [↻] │ ← シャッター行（背景なし）
│   ビデオ  写真       │ ← モードラベル
└──────────────────────┘
```

### ポイント
- プレビューの上下に黒帯がある
- ツールバーボタン（アスペクト比・フラッシュ）は上の黒帯の上に乗っている（黒帯とツールバーは一体化した黒いエリア）
- アスペクト比ボタンを押すとプレビューの画角もそれに応じて変わる
- シャッター行・モードラベルは背景なし
- モードラベル順序: 「ビデオ（左）写真（右）」

## ビデオモード

```
┌──────────────────────┐
│ [HD][30]       [⚡] │ ← バッジ+フラッシュ（透明背景）
│                      │
│                      │
│                      │
│   カメラプレビュー   │ ← 全画面（16:9）
│     （全画面）       │
│                      │
│                      │
│                      │
│     [0.5] [1x]       │ ← ズームプリセット
│ [□] [●シャッター] [↻] │ ← シャッター行
│   ビデオ  写真       │ ← モードラベル
└──────────────────────┘
```

### ポイント
- プレビューが画面全体を埋める
- コントロールはプレビューの上にフロート（オーバーレイ）
- 黒帯なし・背景透明
- 上部に「HD」「30」バッジ（左）とフラッシュボタン（右）

---

## 共通仕様

### ズームプリセットボタン
- 「0.5」「1x」の2つ
- 選択中: 黄色テキスト + 半透明円背景
- 非選択: 白テキスト + 暗い円背景
- タップでズーム倍率が切り替わる

### モード切替
- ラベル順序: 「ビデオ（左）写真（右）」
- スワイプでも切替可能
- 録画中はモード切替・カメラ切替を無効化

### ズームボタンの技術要件
- `sessionQueue.sync` ではなく `sessionQueue.async` を使用（メインスレッドブロック防止）
- `device.minAvailableVideoZoomFactor` を下限として使用（デバイス非対応時の安全なクランプ）
