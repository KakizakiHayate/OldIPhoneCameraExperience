## 更新ルール

このファイルは以下のタイミングで更新すること：

1. **Issue着手時**: ステータスを `🔵 進行中` に変更
2. **PR作成時**: PR # を記入
3. **PRマージ時**: ステータスを `✅ 完了` に変更、テスト数を更新
4. **Phase完了時**: 「現在のフェーズ」セクションを次のPhaseに更新
5. **Issue保留時**: ステータスを `⏸️ 保留` に変更、理由をメモ
